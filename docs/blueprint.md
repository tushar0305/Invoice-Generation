# **App Name**: Saambh Invoice Pro

## Core Features:

- Invoice Creation: Create new invoices with itemized entries (gold/silver weight, stone details, making charges, discounts).
- Invoice Search and Edit: Search for existing invoices and edit their details.
- Printable PDF Generation: Generate printable PDF invoices directly from the app.
- AI-Powered Item Description: Generate descriptions for invoice items based on input keywords using an AI tool.

## Style Guidelines:

- Primary color: Deep maroon (#800000) for elegance and richness, evoking the brand's association with jewellery. This color will appear in headers and other accents.
- Background color: Off-white (#FAF9F6) for a clean and elegant background to contrast the content.
- Accent color: Gold (#D4AF37) as a highlight color to emphasize important actions and information.
- Body and headline font: 'Alegreya', a serif font with an elegant, intellectual and contemporary feel.
- Mobile-first, responsive design for optimal viewing experience on various devices.
- Use minimalist, elegant icons to represent different invoice actions and data points.
- Subtle animations for transitions and user feedback.
